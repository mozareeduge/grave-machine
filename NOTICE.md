# Notice

## Work and authorship

*Grave-Machine* is a bilingual generative electronic poem by Mozare, the artistic name used by Mohammad Zare. It is grounded in Mohammad Zare’s original play *Grave*.

## Taroko Gorge lineage

The work acknowledges Nick Montfort’s *Taroko Gorge* and the literary remix practice that developed around it. The public runtime links to the Electronic Literature Collection presentation:

https://collection.eliterature.org/3/works/taroko-gorge/taroko-gorge.html

This acknowledgement identifies artistic and procedural lineage. It does not claim ownership of *Taroko Gorge* or grant rights beyond those held by the respective authors and publishers.

## Estedad

The Persian interface requests the Estedad variable font from the public Estedad repository through jsDelivr:

https://github.com/aminabedi68/Estedad

Estedad is copyright © 2022 The Estedad Project Authors and is licensed under the SIL Open Font License, Version 1.1. The font file is not stored in this repository; the runtime includes system fallbacks.

## Public/private boundary

The public archive contains the released runtime and public documentation only. Editable project JSON, the bilingual editing workbook, diagnostic output, and the original working-package ZIP are private production materials and are intentionally excluded.
